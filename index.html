方案一：复制粘贴解析

⭐ 最推荐

📋 适用场景

适用于所有WPS和腾讯文档。无需设置权限，无需登录，100%可靠。

🔧 操作步骤

  1. 在WPS或腾讯文档中打开您的在线表格
  2. 用鼠标选中包含数据的列（可以整列选中）
  3. 按 Ctrl+C（Mac: Cmd+C）复制
  4. 粘贴到下方文本框中
  5. 点击「解析数据」按钮

方案二：CSV文件导出后上传

备选方案

📋 适用场景

当数据量较大时适用。在WPS/腾讯文档中导出为CSV文件，然后上传解析。

🔧 操作步骤

  1. 在WPS/腾讯文档中，点击「文件」→「下载」→「CSV」
  2. 保存CSV文件到本地
  3. 点击下方「选择文件」上传CSV文件
  4. 系统自动解析并显示数据

方案三：公开链接API读取

需设置权限

📋 适用场景

仅适用于已设置为「获得链接的人可查看」的公开文档。此方案尝试通过API自动获取数据。

⚠️ 注意：由于浏览器跨域限制(CORS)，此方案成功率有限。WPS文档通常需要登录才能访问数据，推荐使用方案一或方案二。

🔧 操作步骤

  1. 确保文档权限为「获得链接的人可查看」
  2. 在下方输入文档链接
  3. 输入要读取的列名
  4. 点击「尝试读取」
